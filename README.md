# MP3 Player 

Reproductor de música web desarrollado con JavaScript, HTML y CSS.  
Este proyecto incluye demos de música original como parte del proyecto.

--------------------------------------------------------

## Features

- Reproducción canciones y drafts (cuatro en total)
- Interfaz visual personalizada
- Navegación entre tracks
- Integración de imágenes por canción
- Audio dinámico desde archivos locales

--------------------------------------------------------

## Tecnologías

- HTML
- CSS
- JavaScript

--------------------------------------------------------

fragmentos de música original producida por mí:
